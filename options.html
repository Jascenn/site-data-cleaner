🧹 Site Data Cleaner

Manage silent list and view shortcuts.

Silent list

Sites in this list will trigger auto-clear behavior:
• Click the toolbar icon → cleared instantly (no confirmation popup)
• Press F5 / ⌘R → cleared instantly before reload

    Quick reference

    • Toolbar icon (any site) → opens popup with Clear & Reload button
    • Toolbar icon (silent-list site) → instant clear + reload, no popup
    • ⌘+Shift+Y on silent-list site → instant clear + reload
    • ⌘+Shift+Y on other sites → no-op (prevents accidental wipes on Gmail / banks)
    • F5 / ⌘R on silent-list site → intercepted, clear before reload
    • Customize shortcut at chrome://extensions/shortcuts
